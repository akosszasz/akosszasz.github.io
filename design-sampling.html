About Sample Libraries

Creating and designing sample libraries is a passion project of mine. Essentially, these are instruments recorded one note at a time to transform it into a new playable instrument controlled by midi.

Composing these instruments require a lot of attention to detail and with the UI I communicate what mood these libraries evoke or how inspiration came to me.

Sample Libraries

Áferð — Synth

My first actual attempt at creating an instrument was sampling my Korg Minilogue, the UI was clearly inspired by that synth's layout and decided to lean into it, giving it a proper homage.

This is the only sample library I created that got released on Pianobook.

I wanted to explore a way to expand on the synth concept to create an all-in-one instrument for an album of mine.
