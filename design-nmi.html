About Neticle Media Intelligence

Neticle Media Intelligence is the flagship product of Neticle. NMI is a Machine Learning-driven media-monitoring platform turning textual data into measurable information based on keywords and semantic phrases.

Feature Highlights

Modular Chart System

In NMI, the system has three views regarding data visualisation. These are the Report View, the Text Feed View and the Insights View. The challenge was to create a system that would enable cross-referencing elements from these views into their Views as compressed chart-sized modules. The following are examples of these modules.




Annotations + Insight cards

Further exploring the cross-referencing of these modules, we introduced annotations on charts. These nodes represent a peak, a trending topic, anything notable joined by charts further explaining the context of these changes. The insight cards are based on a grid system.


Search Concept

Within the team, we tried to design a context-based predictive search concept. The features we were going for were is having highlights in the auto-complete area, handling typos, keyboard navigation and the context-based entry, which allows the user to create new charts just by typing.
